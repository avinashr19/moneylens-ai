# MoneyLens AI

MoneyLens AI is an AI-powered financial statement analyzer that transforms bank statements into meaningful financial insights.

## Features

- PDF Bank Statement Upload
- Transaction Categorization
- AI Financial Insights
- Interactive Dashboard
- Financial Health Score
- Merchant Analysis
- Income & Expense Reports
- AI Chat with Statements

## Tech Stack

### Backend

- Java 21
- Spring Boot
- PostgreSQL
- Spring AI

### Frontend

- Next.js
- React
- Tailwind CSS
- Recharts
